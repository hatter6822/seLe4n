# scripts/staged_module_allowlist.txt
#
# Per WS-RC R12.B: explicit allowlist of modules that are intentionally
# in the Platform.Staged transitive closure but NOT reachable from the
# production library entry point (SeLe4n.lean).
#
# The companion gate scripts/check_production_staging_partition.sh
# verifies that the live staged_only set equals this allowlist exactly.
# A module that enters production (becomes reachable from SeLe4n.lean)
# without being removed from this allowlist will fail the gate; a new
# staged module added without an allowlist entry will also fail.
#
# Format: <module>  # <category>: <one-line rationale>
# Categories:
#   marker              — file carries a "STATUS: staged for ..." marker
#   transitively-staged — pulled in by a marker-bearing module
#   infra               — platform infrastructure (bridges, anchors, helpers)
#   anchor              — the staging anchor file itself
#
# Last verified: WS-SM SM1.C landing (v0.31.5 HEAD; adds
# `Kernel.SecondaryEntry` to the SM0+SM1.B staged-Concurrency-and-FFI
# closure — the SM1.C.6 Lean entry-point placeholder for
# `rust_secondary_main` enters staged via Platform.Staged).

SeLe4n.Kernel.Architecture.CacheModel         # marker (STATUS: staged for H3)
SeLe4n.Kernel.Architecture.ExceptionModel     # marker (STATUS: staged for H3)
SeLe4n.Kernel.Architecture.InterruptDispatch  # transitively-staged via ExceptionModel
SeLe4n.Kernel.Architecture.TimerModel         # marker (STATUS: staged for H3)
SeLe4n.Kernel.Architecture.TlbCacheComposition # transitively-staged via Staged
SeLe4n.Kernel.Concurrency.Assumptions         # infra (SMP-latent assumption inventory)
SeLe4n.Kernel.Concurrency.Anchors             # infra (WS-SM SM0.C inventory build anchor; SMP-H3 closure)
SeLe4n.Kernel.Concurrency.Locks               # infra (WS-SM SM0.I BklState)
SeLe4n.Kernel.Concurrency.Locks.Kind          # infra (WS-SM SM0.I LockKind + LockId total order)
SeLe4n.Kernel.Concurrency.Sgi                 # infra (WS-SM SM0.H SgiKind)
SeLe4n.Kernel.Concurrency.Runtime             # infra (WS-SM SM1.B.5 currentCoreId FFI wrapper; SMP-M4 Lean side)
SeLe4n.Kernel.SecondaryEntry                  # infra (WS-SM SM1.C.6 secondary-core kernel-entry placeholder; SMP-C2 Lean side)
SeLe4n.Platform.Staged                        # anchor (the staging entry point itself)
